## 1-3. 팩토리얼 문제

~~~ python
a = int(input("팩토리얼 : "))

result = 1
for i in range(a+1):
    if i == 0:
        continue
    
    print("{} * {}".format(result, i))
    result *= i
    
print(result)

~~~
